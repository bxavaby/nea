ntr (read enter)


[WIP] ntr is what I call my current Void Linux setup on my ThinkPad X220. It aims to be very minimalistic and efficient.
If you think I can improve it, or simply want to suggest something cool, please do so via matrix or email.

Setup, CUZ IT VITAL

# Enable graphics for X220 iGPU
xbps-install -S linux-firmware-intel xf86-video-intel mesa-dri

# Specific kernel params (limine.cfg later)
echo 'i915.enable_psr=0 i915.fastboot=1' >> /etc/modprobe.d/i915.conf
